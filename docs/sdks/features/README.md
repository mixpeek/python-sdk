## list_extractors

List all available feature extractors grouped by category

### Example Usage

```python
from mixpeek import Mixpeek
import os


with Mixpeek(
    token=os.getenv("MIXPEEK_TOKEN", ""),
) as m_client:

    res = m_client.features.list_extractors()

    # Handle response
    print(res)

```
