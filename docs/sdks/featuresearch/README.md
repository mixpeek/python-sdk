# FeatureSearch
(*feature_search*)

## Overview

### Available Operations

* [search_features_v1_features_search_post](#search_features_v1_features_search_post) - Search Features

## search_features_v1_features_search_post

This endpoint allows you to search features.

### Example Usage

```python
import mixpeek
from mixpeek import Mixpeek
import os

with Mixpeek(
    token=os.getenv("MIXPEEK_TOKEN", ""),
) as mixpeek:

    res = mixpeek.feature_search.search_features_v1_features_search_post(collections=[
        "collection1",
        "collection2",
    ], offset_position=0, page_size=10, queries=[

    ], filters={
        "case_sensitive": True,
        "and_": [

        ],
        "or_": [

        ],
        "nor": [

        ],
    }, group_by={
        "field": "asset_id",
        "max_features": 10,
        "sort": {
            "field": "metadata.field_name",
            "direction": mixpeek.Direction.DESC,
        },
    }, sort={
        "field": "score",
        "direction": mixpeek.Direction.DESC,
    }, select=[
        "title",
        "content",
        "metadata.author",
        "metadata.publication_date",
    ], reranking_options={
        "weights": {
            "feedback": 0.7,
            "popularity": 0.3,
        },
        "enable_reranking": False,
    }, session_id="sess_abc123", return_url=True)

    # Handle response
    print(res)

```
