# SessionStatus

Session lifecycle states.  Attributes:     ACTIVE: Session is actively processing messages     IDLE: Session exists but no recent activity     ARCHIVED: Session archived (read-only)     TERMINATED: Session permanently closed

## Enum

* `ACTIVE` (value: `'active'`)

* `IDLE` (value: `'idle'`)

* `ARCHIVED` (value: `'archived'`)

* `TERMINATED` (value: `'terminated'`)
