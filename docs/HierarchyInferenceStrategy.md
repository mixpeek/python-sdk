# HierarchyInferenceStrategy

Strategy for inferring the base hierarchy structure.  Can be combined with manual overrides via hierarchical_nodes for hybrid configuration: - SCHEMA: Infer based on overlapping collection schemas - CLUSTER: Infer based on clustering algorithms and overlap detection - LLM: Infer using AI/language models

## Enum

* `SCHEMA` (value: `'schema'`)

* `CLUSTER` (value: `'cluster'`)

* `LLM` (value: `'llm'`)
