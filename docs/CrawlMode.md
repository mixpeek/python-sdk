# CrawlMode

Mode for crawling web pages.  Values:     DETERMINISTIC: BFS crawl following all links up to max_depth     SEMANTIC: LLM-guided crawl prioritizing pages relevant to crawl_goal

## Enum

* `DETERMINISTIC` (value: `'deterministic'`)

* `SEMANTIC` (value: `'semantic'`)
