# VectorPurpose

Semantic purpose of a vector index.  Used by pipelines to look up vector configs by purpose without needing to know the specific inference_service_id.

* `CODE` (value: `'code'`)

* `IMAGE` (value: `'image'`)

* `MULTIMODAL` (value: `'multimodal'`)

* `VIDEO` (value: `'video'`)

* `AUDIO` (value: `'audio'`)

* `SPARSE` (value: `'sparse'`)
