# ParameterType

The type of a feature extractor parameter.  - 'fixed': The value is provided directly. - 'dynamic': The value is resolved from the input data via mapping.

## Enum

* `FIXED` (value: `'fixed'`)

* `DYNAMIC` (value: `'dynamic'`)
