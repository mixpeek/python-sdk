# RenderStrategy

Strategy for rendering web pages.  Values:     STATIC: Fast HTTP fetch, works for most sites     JAVASCRIPT: Browser rendering via Playwright for SPAs     AUTO: Try static first, fall back to JS if content too short

## Enum

* `STATIC` (value: `'static'`)

* `JAVASCRIPT` (value: `'javascript'`)

* `AUTO` (value: `'auto'`)
